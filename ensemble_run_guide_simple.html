Guidelines to Run the Ensemble Code

1. Data Preparation

Run:

./Input/Input Data And Cleaning For The Model.R

2. Model Execution

Run these scripts, up to the latest available date.test2 (forecasting dates):

./Model/R/01_call_inla_spacetime.R
./Model/R/02_call_hhh4.R

3. Baseline Model

Run:

./Model/R/baseline_inla_method.R

Then:

./Output/Build_the_ensemble.R
./Output/fun_threshold_risk_scores.R

4. Generate Outputs

Run:

./Output/Generate_output_summary_2025.R
./Output/find_outbreaks.R
